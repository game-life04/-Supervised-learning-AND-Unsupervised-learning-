### Unsupervised learning

Data only comes with inputs x,but not output lables y.(数据仅包含输入变量x，但不包含输出标签y)
Algorithm has to find ==structure== in the data(算法必须在数据中找到结构)

1:Clustering：Group similar data points together(1:聚类：将相似的数据点分组)
2:Anomaly detection：Find unusual data points(2:异常检测：发现异常数据点)
3:Dimensionality reduction：Compress data using fewer numbers.(3:学习降维：使用更少的数字压缩数据)
